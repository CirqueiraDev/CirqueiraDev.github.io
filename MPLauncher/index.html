Why Choose MPLauncher

Everything you need.
Nothing you don't.

Built for players who want simplicity, control, and offline freedom.

Offline Mode

No Microsoft account required. Play completely offline using any nickname — perfect for LAN sessions and cracked usage.

Auto Version Install

Fetches the full Mojang version list and installs any release, snapshot, beta or alpha automatically — with real-time progress feedback.

Live Console Output

Real-time log streaming during both download and gameplay. See every JVM message as it happens, right inside the launcher.

RAM & JVM Control

Adjust memory allocation with a slider and pass custom JVM arguments. Pre-tuned G1GC flags ship out of the box for smoother gameplay.

Custom Skin Support

Load your own skin PNG and it appears in-game — a built-in local skin server serves the texture directly to Minecraft without any external service.

Java Detection

Automatically detects your Java installation and reports its version. Clear error messages guide you if Java is missing or misconfigured.

About the Project

Simple, open,
and offline-first.

MPLauncher started as a weekend experiment to build a clean, minimal Minecraft launcher that requires no Microsoft account. It uses minecraft-launcher-lib under the hood, a PyQt5 interface, and ships with sane defaults for low-spec machines. Every setting is persisted in a single settings.json file — no registry, no telemetry.

Python 3.10+ PyQt5 5.15+ Windows / macOS / Linux No Telemetry

Configuration

Persistent settings.
Zero friction.

All preferences are saved automatically in settings.json at the project root.

minecraft_directory Custom path to your .minecraft folder
ram_mb RAM allocation in MB — default 2048
jvm_args Extra JVM flags appended at launch
show_snapshots Show snapshot, beta and alpha versions
close_on_launch Minimize launcher when game starts
last_nickname Saves your nickname between sessions
